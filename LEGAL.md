# 法務メモ — 「ITパス理解ノート」を安全に運営するために

> このドキュメントは公式な法律意見ではありません。最終的な判断は弁護士に相談してください。

## 1. IPAの公式ポリシー（重要）

[IPA「試験についてのよくある質問」](https://www.ipa.go.jp/shiken/faq.html) で公開されている公式見解の要旨:

- 当機構で公表している過去の試験問題の使用に関し、許諾や使用料は **不要**
- ダウンロードでの利用も **可**
- ただし留意点を必ず守ること

つまり **教育目的なら許諾なしで使える**。守るべきは以下の5つ。

### 守るべき5つのルール

1. **著作権はIPAに帰属する** — 自分のものとして扱わない
2. **教育目的の範囲内** — 情報処理技術者試験制度の意義に反しない用途のみ
3. **出典を明記** — 指定形式に従う（年度、期、試験区分、時間区分、問番号）
4. **改変したら明示** — 一部でも変更したら「※一部改変」等を明記
5. **PDFそのものは再配布しない** — 公開PDF以外の電子データ提供は不可、PDFセキュリティ解除も不可

### 本サイトの遵守状況

| ルール | 対応 |
|---|---|
| 著作権はIPA帰属 | ✅ AttributionNote と /legal で明示 |
| 教育目的 | ✅ 学習補助ツールとして運用 |
| 出典明記 | ✅ 各問題に IPA指定形式で表示 |
| 改変明示 | ✅ `modifiedNote` で「※OCR起因の表記揺れを統一」等を明示 |
| PDF非再配布 | ✅ 元PDFは `.gitignore` で公開リポジトリに含めない |

## 2. ITパスポート試験の引用書式

IPAが指定する出典表記の形式は「年度、期、試験区分、時間区分、問番号」。
ITパスポートはCBT方式で「春期/秋期」の区別がない年度がある。本サイトでの基本書式:

```
出典：令和7年度 ITパスポート試験 公開問題 問1
```

## 3. 何ができて、何ができないか

### ✅ できる
- 公表済み過去問題を **問題集・テキスト・Webサイト・アプリ** に使用
- 問題文・選択肢・正解・公式解説の引用
- ダウンロード・取り込み（手元利用）
- 教育目的での無償公開

### ⚠️ 注意して扱う
- 商用利用 — 「教育目的」の範囲なら可。ただし IPA に問い合わせれば確実
- 広告掲載 — 教材サイトでの広告は一般的にOKだが念のため確認推奨
- 大量転載 — IPAの公式FAQには「量」の制限なし。ただし試験制度の意義を損なう使い方はNG

### ❌ できない
- IPA配布のPDFそのものを再配布
- PDFセキュリティ解除方法の提供・公開
- 出典を書かずに使う
- 改変したのに「変えてない」かのように見せる

## 4. 改変の扱い

本サイトの過去問は、スキャンPDFをOCR (Tesseract) で読み取って構造化したもの。OCR起因で生じる以下の修正は、IPA の「改変」に該当する可能性があるため `modifiedNote` で明示:

- OCR誤認識の文字修正
- 全角/半角の統一
- 改行位置の調整
- 句読点・記号の表記揺れ統一

問題の **意味・選択肢の内容・正解** は一切変更していない。

## 5. 解説・分析の著作権

本サイト独自の創作物（IPAの著作物ではない）:
- 解説本文 (`explanation`)
- 「なぜこの選択肢が魅力的に見えたか」(`whyAttractive`)
- 「押さえるコツ」(`keyInsight`)
- 「典型的な間違え方の流れ」(`commonMistakeFlow`)
- 誤解パターン辞書 (`misconceptions`)
- 補助資料・比較表 (`materials`)
- UI・デザイン・ロジック

これらは本サイトの著作物として、無断転載・複製を禁ずる旨を /legal で明示。

## 6. オリジナル問題の扱い

過去問とは別に、IPA出題範囲を元に独自に作成した問題 (`originType: "ipa_inspired"`) を併載している。これは本サイトの創作物で、出典は不要。UI上「✨ オリジナル」バッジで区別。

## 7. プライバシー

ユーザの学習履歴は `cookie` ベースの匿名IDで管理。氏名・メールアドレス等の個人情報は一切収集しない。
将来ログイン機能を追加する場合は、個人情報保護法に従ってプライバシーポリシーを整備する。

## 8. 削除依頼・問い合わせ窓口

`/legal` ページ末尾に GitHub Issues / メールアドレスを掲載。

## 9. 商用化を視野に入れる場合

- 教育目的の範囲を超える可能性が出てきたら **念のため IPA に事前確認**
  - 窓口: https://www.ipa.go.jp/shiken/inquiry.html
- 利用規約・プライバシーポリシーを整備
