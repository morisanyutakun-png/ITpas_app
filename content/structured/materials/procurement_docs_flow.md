## 段階別フロー

```
[RFI] → [RFP] → [RFQ] → [契約]
情報収集  提案依頼  見積依頼   発注
```

| 文書 | 略称の元 | 何を依頼するか | 段階 |
|---|---|---|---|
| **RFI** | Request for **Information** | 活用事例・技術情報の提供 | 情報収集 |
| **RFP** | Request for **Proposal** | 要件に対する**提案書** | ベンダ選定 |
| **RFQ** | Request for **Quotation** | **見積書** | 価格確定 |

## 紛らわしい略語（調達ではない）

| 略語 | 意味 |
|---|---|
| EDI | 取引データの電子交換 |
| KPI | 重要業績評価指標 |
| SLA | サービスレベル合意 |

## 試験頻出のひっかけ

- 「活用事例や技術情報の提供を依頼する文書」 = **RFI** （RFPではない）
- 順序は「**I → P → Q**」
