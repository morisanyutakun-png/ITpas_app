| 書き方 | コピーで動く？ | 用途 |
|---|---|---|
| `A1` | 行・列ともにズレる | 通常の相対参照 |
| `$A$1` | どちらもズレない | 完全固定（定数を参照する時） |
| `$A1` | 列は固定、行はズレる | 縦方向にコピーする時、列を固定 |
| `A$1` | 行は固定、列はズレる | 横方向にコピーする時、行を固定 |

## 主要関数

| 関数 | 意味 |
|---|---|
| `IF(条件, 真, 偽)` | 条件分岐 |
| `SUM(範囲)` | 合計 |
| `AVERAGE(範囲)` | 平均 |
| `COUNT(範囲)` | 数値の件数 |
| `COUNTIF(範囲, 条件)` | 条件に合う件数 |
| `VLOOKUP(検索値, 範囲, 列番号, 0)` | 完全一致検索 |
| `IFERROR(値, エラー時)` | エラー処理 |

## ひっかけ

- VLOOKUP の第4引数は `0`/`FALSE` で**完全一致**（試験で頻出）
